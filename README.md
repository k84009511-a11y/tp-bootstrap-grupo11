##  Descripción del proyecto

El proyecto consiste en el desarrollo de un sitio web informativo sobre el turismo en la ciudad de Formosa.

El sitio está compuesto por cuatro páginas principales:

- Página de inicio (portal de bienvenida)
- Página de alojamientos
- Página de excursiones y tours
- Página de contacto (consultas, horarios de atención y ubicación)

El objetivo del sitio es brindar información clara y accesible sobre lugares turísticos, hospedajes y actividades disponibles en la ciudad.

---

## Componentes de Bootstrap utilizados

- Navbar responsivo  
- Sistema de grillas (Grid System)  
- Cards para mostrar información de hoteles y servicios  
- Formulario de contacto  
- Botones estilizados  
- Footer con estructura en columnas  
- Carrusel de imágenes con autoplay
